# Basic testing

This task is a part of [RSSchool Node.js Course](https://rs.school/courses/nodejs) 2005.

[Task description](https://github.com/AlreadyBored/basic-testing)

---

### Installation

1. Install [Node.js](https://nodejs.org/en/download/)
2. Clone this repository
3. Switch to `dev` branch
4. To install all dependencies use [`npm install`](https://docs.npmjs.com/cli/install)
5. Run **test scripts** in command line.
6. You will see the number of skipped, passing and failing tests.

---

### Test scripts

```bash
# run unit tests
$ npm run test

# with logging
$ npm run test:verbose
```
